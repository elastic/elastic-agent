The 0.1.0 release adds the following new and notable features.




* Add sample config files for Windows ES and mOTLP ingestion. https://github.com/elastic/elastic-agent/pull/10728[#10728] https://github.com/elastic/elastic-agent/issues/10540[#10540]
* Added opex to elastic-agent helm chart, This change will add the Opex-CCM support to the offical elastic-agent helm chart deployment.  


* Add awss3receiver. https://github.com/elastic/elastic-agent/pull/10515[#10515] https://github.com/elastic/obs-integration-team/issues/604[#604]
